# 2 минуты
Мобильное приложение для коротких тренировок — физических и ментальных упражнений, которые можно выполнить за 2 минуты. Построено на Flutter, работает на iOS и Android.
Идея
Приложение помогает пользователям делать регулярные микро-перерывы в течение дня: разминка шеи, плеч и спины, упражнения для глаз, дыхательные практики и техники переключения внимания. Каждая тренировка состоит из трёх упражнений по 40 секунд.


## Возможности

Авторизация — вход, регистрация, восстановление пароля - будет реализовано
Онбординг — ввод имени, выбор проблемных зон, настройка времени и частоты уведомлений
Главный экран — приветствие по имени и времени суток, быстрый старт тренировки
Выбор упражнений — ручной выбор из каталога (физические / ментальные) или случайный подбор
Тренировка — таймер с круговым прогрессом, пауза, пропуск, регулировка времени ±15 сек, добавление в избранное
Каталог — все упражнения по категориям с описаниями (шея, плечи и руки, спина и поясница, глаза, снятие напряжения, переключение внимания, эмоциональная стабилизация)
Настройки — программа, уведомления, язык, внешний вид, премиум - будет реализовано
Нижняя навигация — Главная, Каталог, Настройки

##Стек

Flutter 3.35+, Dart 3.9+
SQLite (sqflite) — хранение каталога упражнений
SharedPreferences — пользовательские настройки и данные онбординга
Provider — управление состоянием
